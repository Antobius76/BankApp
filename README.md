# BankApp. Виджет банковсковского приложения

+ Протграмма создана для работы клиентов банка с банковскими счетами и картами

  ## Функции проекта:
  + Функция маскировки номера карты и счета
  + Функция сортировки по дате

## Зависимости проекта:
+ Python 3.12.4
+ flake8 7.1.0
+ black 24.4.2
+ isort 5.13.2
+ mypy 1.10.0
